## Общее описание

Данный репозиторий содержит 5 лабораторных работ, посвященных практическому изучению и реализации классических алгоритмов машинного обучения. Каждая работа следует единой структуре: от построения базовой модели (бейзлайна) до самостоятельной имплементации алгоритма и сравнения его производительности с библиотечными аналогами.


В качестве наборов данных использовались:
-   **Классификация:** [Credit Card Customers](https://www.kaggle.com/datasets/sakshigoyal7/credit-card-customers) (прогнозирование оттока клиентов банка).
-   **Регрессия:** [Flight Fare Prediction](https://www.kaggle.com/datasets/nikhilmittal/flight-fare-prediction-mh) (прогнозирование цен на авиабилеты).

## Содержание

-   Лабораторная работа №1: K-ближайших соседей (KNN)
-   Лабораторная работа №2: Линейные модели
-   Лабораторная работа №3: Решающее дерево
-   Лабораторная работа №4: Случайный лес
-   Лабораторная работа №5: Градиентный бустинг и итоговые выводы
